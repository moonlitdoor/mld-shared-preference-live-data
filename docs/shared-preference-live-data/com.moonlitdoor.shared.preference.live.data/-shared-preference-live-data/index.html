SharedPreferenceLiveData

abstract class SharedPreferenceLiveData<T> : LiveData<T>, OnSharedPreferenceChangeListener

SharedPreferenceLiveData is a generic data holder class that can be observed within a given lifecycle which is backed by a shared preference.

key - The shared preference key.
